# Gerador-QR-code
Gennera e Youtube. 
Projeto Gerador QR Code desenvolvido com HTML, CSS e JavaScript. Permite criar códigos QR de forma rápida e prática a partir de textos, links ou informações digitadas pelo usuário. Possui interface simples, responsiva e intuitiva, oferecendo uma experiência eficiente para gerar e compartilhar QR Codes.
